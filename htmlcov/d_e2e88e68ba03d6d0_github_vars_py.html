6class GithubVars(DocumentedEnvVars): 

7 """GitHub Action environment variables. 

8 

9 https://docs.github.com/en/actions/learn-github-actions/variables#default-environment-variables 

10 

11 Usage: 

12 class MyAction: 

13 @property 

14 def vars(self): 

15 return GithubVars() 

16 

17 action = MyAction() 

18 print(action.vars.github_repository) 

19 

20 Thanks to the docstrings your IDE will provide you with doc hints when you hover over the property. 

21 We do not load the attributes on the class init but do it Lazily. 

22 Once read, the value is stored in the instance dictionary and is not extracted from env anymore. 

23 

24 Only described Github vars are loaded. 

25 Paths and files have type Path. 

26 """

31 github_action: str 

32 """The name of the action currently running, or the id of a step. 

33 For example, for an action, __repo-owner_name-of-action-repo. 

34 GitHub removes special characters, and uses the name __run when the current step runs a script 

35 without an id. If you use the same script or action more than once in the same job, the name 

36 will include a suffix that consists of the sequence number preceded by an underscore. For 

37 example, the first script you run will have the name __run, and the second script will be 

38 named __run_2. Similarly, the second invocation of actions/checkout will be actionscheckout2."""

107 """The fully-formed ref of the branch or tag that triggered the workflow run. For workflows
